# Rock Paper Scissors Game (C++)

A simple console-based Rock Paper Scissors game written in C++.

## Features
- Player vs Computer
- Multiple rounds (1 - 10)
- Shows round results
- Displays final game results
- Replay option

## Concepts Used
- Functions
- Enums
- Structs
- Random numbers
- Functional programming style

## How to Run

1. Compile the program: 
g++ main.cpp -o game

2. Run the program: 
./game
